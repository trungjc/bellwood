Installation

Requirements

Verona theme requires WordPress 3.4 or greater. You must have WordPress already installed on your hosting. If you are not familiar of how to install WordPress please procede to this link "How to install WordPress"

How to install Verona and activate it

Upload your theme:

1. Go to your WordPress admin dashboard, on the left panel navigate to Appearance > Themes

2. Click the Install Themes tab, and after that click on Upload link

3. Click on Choose File and select the already extracted verona.zip file from the downloaded package

4. Click the Install Now button 

5. That's it, Verona theme has been uploaded to your WordPress.

Activate your theme:

1. After confirmation of installation, navigate to Appearance > Themes > Manage Themes

2. Locate the Verona theme and below the theme screenshot click on Activate

3. And Voila, now Verona theme is activated and running.

Activate the premium plugins:

After activation you will be prompted to install the included premium plugins into your theme, the LayerSlider and the Symple Shortcodes plugins that are perfectly integrated to work flawlessly with our theme.

Import the Demo Data

To import the demo data all you have to do is navigate in your WordPress dashboard to Tools --> Import --> WordPress --> and click Install Now after the installing of the plugin WordPress Importer activate the plugin by clicking on the link Activate Plugin & Run Importer

Second step is to import the xml file that is in DemoData folder in your already downloaded Verona Package.

To do so click on Choose file and import the xml file that is in the DemoData folder.  After the upload of the file be sure to include everything, including attachments.

And that's it now you have all the content same as the live demo ready and imported into your new installation. (All posts, pages, menus, images, food and coffee menus etc).

Import the LayerSlider Demo Data - optional

Since version 1.4 we have included also the LayerSlider demo data, to import it into your LayerSlider plugin simply open the file named layerslider-demo-data.txt and open it with text editor, copy the whole text and paste it into the LayerSlider import textarea. Click the Import button and the sliders will be imported.

Pages & Navigation

Pages

Create a New Page:

1. Go to the left panel and navigate to Pages > Add New

2. Enter the title for the new page, for e.g Home Page

Choose a page template for your page:

Under page attributes you can select under what template this page to be displayed, or simply what layout to use.

General Page Options:

Under page options you can choose to:

1. Choose to display or not the tagline (for example you don't want to delete the tagline text but you also don't want or need to display it on the frontend)

2. Disable the page title but keep the tagline text

3. Write and add your tagline text

Food Page Options:

1. If this is going to be a food menu page, than you have the option of choosing which already created food categories and their items to be displayed. You can also leave it empty so all food menu items from all categories will be shown. Make sure you have selected the Food Menu template.

Coffee Page Options:

1. If this is going to be a coffee menu page, than you have the option of choosing which already created categories and their items to be displayed. You can also leave it empty so all food menu items from all categories will be shown. Make sure you have selected the Coffee Menu template for this page.

2. Option to select the grid layout for this particular coffee page.

LayerSlider Options:

We have made it easy for you to include different slider on each separate page, first of all to use this feature you must already have created slider with slides with the LayerSlider plugin:

1. Enable or Disable the slider (every time you create a new page, the layerslider is disabled by default)

2. Choose which already created slider you want to display on the desired page by simply choosing it's name from the dropdown options

Menus

Verona uses three menus:

1. Left Header Navigation Menu (assigned left to the logo)

2. Right Header Navigation Menu (assigned right to the logo)

3. Footer Navigation Menu (displayed at the bottom of the page, right to the copyright notice, if is disabled, the copyright notice is centered)

Shortcodes of Verona

Below are all shortcodes that exists in Verona Theme and their parameters briefly explained, listed alphabetically:
 
-- Accordions --

The accordion shortcode has two parameters, accordion title and accordion content


[accordion][accordion_section title="Section 1"]Accordion Content[/accordion_section][/accordion]

Accordion parameters:

title="" -- the title of the accordion

-- Alerts --

The alert shortcode has one parameter with 4 options and alert content


[alert type="error-notice-info-success"]Your Content displayed in the alert box[/alert]

Alert parameters:

type="" -- there are four types of alerts which are error notice info success

-- Band --

The band shortcode act as a full width row, simply cuts and gets out of the page making it's own full width row.


[band background_color="#150803" background_image="http://yoururltotheimage.com/image.jpg" type=""]Put your content text here or put new shortcodes[/band]

Band parameters:

background_color="" -- your custom color

background_image="" -- here goes the full path url of your image that will stretch full width, the band automatically fits the height of the image

-- Box --

The box shortcode act as a row, you always put columns in it.


[box][/box]

Box parameters:

There are no paramaters for this shortcode, but check below the boxes it's usage.

-- Boxes --

The boxes shortcode is used for you to put a boxed content with title and image or icon, as also a text content

Boxes parameters:

title="" -- your custom title text

image="" -- here goes the full path url of your image or image icon that you want to be displayed

-- Buttons --

Buttons are cute and simple way to add button with link whenever there is need on your page, it has couple of parameters


[button color="#ff7400" url="http://www.suitstheme.com" title="Visit Site" target="blank" border_radius="" arrow="yes"]Sample Content[/button]

Button parameters:

color="" -- the text color of the button

url="" -- the url where this button will redirect to

title="" -- the text title of the button

target="" -- should this button open a new window or stay on the same page after clicking, there are four values: top(Opens the linked document in the full body of the window), blank(Opens the linked document in a new window or tab ), self(Opens the linked document in the same frame as it was clicked (this is default) ), parent(Opens the linked document in the parent frame )

border_radius="" -- if you want your button to have rounded edges set the border radius in pixels, for example 3px

arrow="" -- if you want this button to have a small icon arrow inside set it to yes, for example arrow="yes"

-- Callout --

The callout shortcode outputs a callout banner with a content text and a button.


[callout background_color="#68524A" text_color="#fff" button_text="button text" button_color="blue" button_text_color="#000" button_url="http://www.suitstheme.com" button_rel="nofollow"]Sample Content[/callout]

Boxes parameters:

background_color="" -- the background color of the whole callout

text_color="" -- the content text color

button_text="" -- the text for the button

button_color="" -- background color of the button

button_text_color="" -- the button text color

button_text_color="" -- the linking url of the button

The background color, text color and button text color parameters are added with the release of Verona version 1.4

-- Clear Floats --

This shortcode simply clear all elements on both sides, it inserts a css class with property clear:both;


[clear_floats]

-- Dividers --

Dividers are great addon for you to separate the content on your pages


[divider style="solid" margin_top="20px" margin_bottom="20px"]

Divider parameters:

style="" -- there are 5 styles you can use solid, dashed, dotted, double, fadein and fadeout

margin_top="" -- add how much space would you like to have from the top of the divider, in pixels value for example 15px

margin_bottom="" -- add how much space would you like to have from the bottom of the divider, in pixels value for example 15px

-- Google Map --

With this shortcode you can insert google map on every page you want to


[googlemap title="Envato Office" location="2 Elizabeth St, Melbourne Victoria 3000 Australia" zoom="10" height=250]

Google map parameters:

title=="" -- add title or name for this google map

location="" -- add the location of your targeted place, is simply the address, make sure is correct

zoom="" -- add number so the map is zoomed in or zommed out, if you put higher number the map is more zoomed out

height="" -- add number of the height of your map, without the pixels value, simpy 100 or 400 or whatever suits your map best.

-- Highlights --

With this shortcode you can highlight text


[highlight color="blue"]Sample Content[/highlight]

Highlight parameters:

color="" -- add a predefined color, these are available: black, blue, gray, green, red, yellow and orange

-- Latest Posts from Blog --

This shortcode gets your latest 3 blog posts and display them


[latest_posts fromcategory="" icon="" title="Latest Blog Posts" description="Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium, totam rem aperiam, eaque ipsa quae ab illo." linkurl="#" linkname="See all posts"]

Latest posts parameters:

fromcategory="" -- specify the category name from which this shortcode will grab the latest posts, if you leave it empty will grab the latest blog posts from all categories

icon="" -- here goes the full path url of your image or image icon that you want to be displayed

title="" -- add the title of the shortcode, for example latest from blog, our news, latest news, kitchen news etc.

description="" -- add the description of the shortcode

linkurl="" -- add the url link to the blog page or other page

linkname="" -- add the name of the link, for example see all posts, read more or click for more etc.

-- Popular Menu --

This shortcode gets 3 food menu items and displays them


[popular_menu fromcategory="" icon="" title="Most Popular" prices="show" description="Sed ut perspiciatis unde omnis iste natus error sit voluptatem accusantium doloremque laudantium, totam rem aperiam, eaque ipsa quae ab illo." linkurl="#" linkname="See all menu"]

Popular menu parameters:

fromcategory="" -- specify the category name from food menu custom type and it will grab the posts from the specified category, if you leave it empty will grab all latest added food menu items from all categories, for example you can create popular category under food menu custom type and add there items and later just specify that category in this parameter

icon="" -- here goes the full path url of your image or image icon that you want to be displayed

title="" -- add the title of the shortcode, for example our specials, popular dishes, latest menu etc.

prices="" -- you can show or do not show the prices for the food menu items, by putting prices="show" will show the prices, if you leave it empty it won't show the prices (this attribute has been added to Verona version 1.4)

description="" -- add the description of the shortcode

linkurl="" -- add the url link to the food menu page or other page

linkname="" -- add the name of the link, for example see all dishes, read more or see full menu etc.

Translation

Verona theme is a translation ready, that means you can easily translate to your native language.

You will need the POEdit tool to translate all the strings from Verona theme, you can download the software from HERE,  after you have downloaded and installed the software, open the english.po file from the includes/languages folder and translate the texts.

Save it in the same folder with a new name or simply overwrite the english one (only if you don't need the english language). If you save it under new name let's say you translate it to german and you saved it as de_DE, than go to the root of your WordPress installation and open the wp-config.php file find the line with define('WPLANG', ''); and modify it like this define('WPLANG', 'de_DE'); 

And that's it, the Verona theme is translated.
